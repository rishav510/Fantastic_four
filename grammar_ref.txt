
<ID>               --> [A-Za-z_][A-Za-z0-9_]*

<STMTS>            --> <DECLARE_STMTS> <ASSIGNMENT_STMTS>
<DECLARE_STMTS>    --> <DECLARE_STMT> <DECLARE_STMTS>
<ASSIGNMENT_STMTS> --> <ASSIGNMENT_STMT> <ASSIGNMENT_STMTS>

<DECLARE_STMT>     --> declare <ID> : <TYPE> ; | declare list of variables <IDS> : <TYPE> ; 
<IDS>              --> <ID> | <ID> <IDS>






<ASSIGNMENT_STMT>  --> <ID> = <ARITHMETIC_EXPR> | <BOOLEAN_EXPR> ;

<ARITHMETIC_EXPR>  --> <ARITHMETIC_EXPR> + <LINEAR_EXPR> | <ARITHMETIC_EXPR> - <LINEAR_EXPR> | <LINEAR_EXPR>
<LINEAR_EXPR>      --> <LINEAR_EXPR> * <FACTOR> | <LINEAR_EXPR> / <FACTOR> | <FACTOR>
<FACTOR>           --> <VAR> | (<ARITMETIC_EXPR>) | <NUM> 
<VAR>              --> <ID> | <ARRAY_ELEMENT> 
<ARRAY_ELEMENT>    --> <ID> [ <LIST_INDICES> ]
<LIST_INDICES>     --> <NUM> <LIST_INDICES> | <ID> <LIST_INDICES>
<NUM>              --> [0-9]+


<BOOLEAN_EXPR>     --> <BOOLEAN_EXPR> ||| <OR_EXPR>
<OR_EXPR>          --> <OR_EXPR> &&& <AND_EXPR>
<AND_EXPR>         --> <ID>


<ArrayDecl> --> <SingleValArray> | <MultivalArray>
<MultivalArray> --> declare list of variables <ID><IDS> : array  <DIM > of integer <Semicolumn>
<IDS> --> <ID><List> | <ID>
<DIM> --> SQO low..high SQC <DIM> | SQO low..high SQC 
<SingleValArray> --> declare <ID> : array <DIM> of integer <Semicolumn>
<Semicolumn> --> ';'

//Grammar for jagged Array

<Declaration>--><ArrayDecl> <semicolon> <R1_Array>
<ArrayDecl> --> <SingleValjaggedArray> | <MultivaljaggedArray>
<MultivaljaggedArray> --> declare list of variables <List> : jagged array  <DIM > of integer
<SingleValjaggedArray> --> declare <ID> : jagged array <DIM> of integer
<R1_Array> -->R1 SQO <integer> SQC : size <Integer> : values CBO <NUM> CBC <R1_Array>| Ø
<List> --> <ID> space <List> | <ID>
<DIM> --> SQO low..high SQC <DIM> | SQO SQC <DIM> | Ø
<NUM> --><ID1> <Semicolon> <NUM>| <integer>
<ID1> --><Integer> space <ID1> | <integer>
<Semicolon> --> ';'
